# ctrlZ
제로웨이스트 습관 형성 플랫폼 Ctrl Z는 사용자의 성향테스트를 통해 사용자들에게 추천 챌린지와 미션을 제공하고 데일리미션을 통해 습관을 형성할 수 있습니다. 사용자들끼리 미션을 수행하고 공유하며 커뮤니티를 형성합니다.
